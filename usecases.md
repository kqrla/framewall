# use cases

## who is this for

### museum app developers
building a virtual museum, gallery app, or exhibit viewer. framewall generates the architectural backdrop — walls, wainscoting, frames — so exhibits feel like they're in an actual museum rather than floating in a void. different wall combinations instantly evoke different museum types:

- victorian gallery → burgundy damask + beadboard + chair rail + baroque gold frames
- modern art museum → gallery white paint + no wainscoting + thin black frames
- natural history museum → olive paint + stone dado + arts & crafts oak frames
- european palace → gold brocade + raised panel + ornate rococo frames

### game designers
creating museum, gallery, or historic interior environments. framewall outputs can serve as background layers, environment textures, or reference material for 3d modelers. the composable wall system means you can generate unique rooms without hand-painting each one.

### graphic designers
need realistic museum wall mockups for:
- presenting artwork in context (gallery-style portfolio presentations)
- creating invitation designs for exhibitions
- designing museum signage and wayfinding
- building pitch decks for cultural institutions

### figma designers
working on museum-related UI/UX projects. the plugin drops frames and walls directly onto the figma canvas as editable vector nodes — every rectangle, ellipse, and shape is individually selectable and modifiable.

### educators & art historians
teaching about frame styles, architectural periods, or museum design. the anatomy-first approach makes it easy to show students how a baroque frame differs from a neoclassical one by toggling individual parameters rather than swapping entire assets.

### interior designers
exploring period-accurate wall treatment combinations for residential or commercial spaces. the composable system covers the same vocabulary used in real architectural millwork: beadboard, raised panel, board and batten, chair rail, picture rail, molding profiles.

## example workflows

### virtual gallery for an artist portfolio
1. open the web app
2. set background mode
3. choose painted wall, museum red
4. enable chair rail divider
5. enable raised panel lower wall in ivory
6. export as png at 1778x1000
7. in your portfolio site, layer artwork images on top of the wall background

### museum app room generation
1. define 5 gallery "themes" using different wall combinations
2. generate each wall background
3. switch to frame mode
4. generate period-matched frames for each room
5. export as svg for scalable integration into the app

### figma design system for a museum
1. open the figma plugin
2. generate a set of wall backgrounds for different gallery types
3. generate frames in each historical style
4. use the outputs as component backing for a museum design system
5. mix and match walls and frames across mockup screens

### art history lecture materials
1. open the web app
2. switch to frame mode
3. start with the baroque preset — show the profile stack, ornament choices, finish
4. modify one parameter at a time to show how styles differ
5. export each variation as png for slide deck

### interior design mood boards
1. generate 3-4 wall variations: damask + beadboard, painted + raised panel, fabric + marble
2. export each as png
3. place in figma or illustrator alongside furniture mockups and color swatches
4. use the frame generator to create matching picture frame styles

## what you get from each export format

| format | best for |
|--------|----------|
| png | mockups, backgrounds, presentations, social media |
| jpeg | web assets where file size matters, photo composites |
| svg | scalable assets for apps, print, or further editing in vector tools |
| html/css | embedding in web pages, interactive museum sites, web-based gallery apps |
